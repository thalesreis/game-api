## Installation
```bash
$ yarn install
```

## Test
```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```

## Running the app - dev env
```bash
# start database
$ yarn db:dev:up

# apply migrations
$ yarn prisma:dev:deploy

# start development-mode API
$ yarn start:dev
```

[Go to http://localhost:3000/api/ and have fun!](http://localhost:3000/api/)
